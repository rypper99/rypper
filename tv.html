Understanding Hisense models

  • first letter is followed by a number to indicate where in the lineup it stands
  • the last letter represents the year
    • G is 2021
    • H is 2022
    • K is 2023
  • U8H vs A6H
    • U8H is high-end 2022 model (year H)
    • A6H is entry-level 2022 model (year H)
  • example: 55A9GTUK
    • 55 = screen size, in inches
    • A = Product line
    • 9 = Product tier
      • single numbers denote high-end models
      • whereas double numbers represent the model position in the hierarchy, counting downwards
    • G = Year, G representing 2021
    • Q = Quantum Dot Colour Technology
    • TUK = Sale region

Understanding Samsung Model Numbers

  • U N 55 TU690T F X Z C
    • U - LED (type of TV) - others Q=QLED, L=LCD, K=OLED
    • N - North America
    • 55 - screen size
    • TU690T - model name
    • F = ATSC tuner type (US, Canada)
    • X - design code
    • ZC - made for Canada (ZA - made for USA)
