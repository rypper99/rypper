The 5 Best Free Screen Recording Apps Without Time Limits or Watermarks

Create screen recording (MOV) using Quick Time player
  • change Sound Output to go to Screen recording audio
    • open System Settings
    • open Sound
    • under Output choose Screen recording audio
  • Starting recording
    • Start Quicktime
      • open QuickTime Player + choose File > New Screen Recording -or-
      • shift + command + 5
    • click on Options
      • choose Quick Time input (I had to choose BlackHole 16ch)
    • click Record to start recording
  • Stop recording
    • click Stop button (at top of screen) to stop recording
  • revert Sound Output back to normal (so can use volume keys on keyboard)
    • open System Settings
    • open Sound
    • under Output choose External Headphones or Mac mini Speakers
Convert MOV to MP4 - using Handbrake
  • open Handbrake
  • open MOV file
  • recommended settings
    • Preset: Fast 1080p30
    • Video tab
      • Video Encoder: H.264 (x264)
      • Framerate (FPS): 30 and select Constant Framerate
        • note: Peak Framerate (aka variable framerate) is known to cause issues
    • Audio tab
      • Codec: AAC (avcodec)
  • Choose save location of output
    • click on Browse (lower right) to pick save location
  • Start Encode
    • click Start Encode button at top of the screen
Perform Screen Recording - Installation and Configuration of BlackHole plugin
  • followed: How to Screen Record with Internal Audio on QuickTime Player (YouTube)
  • download plugin BlackHole : find Download Installer
  • run installer
    • choose BlackHole 16ch option
  • open Audio MIDI Setup (can use search button)
    • will see: BlackHole 16ch
    • create Quick Time Input
      • + (Create Aggregate Device)
      • select BlackHole 16ch
      • double-click on Aggregate Device name and rename => Quick Time Input
    • Create Screen recording audio
      • + (Create Multi-Output Device)
      • select Built-in Output (first)
      • select BlackHole 16ch (second)
      • double-click on Multi-Output Device name and rename => Screen recording audio
